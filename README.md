# Agenda

Agenda is a simple and clean task management tool. The source code is available here on GitHub.

## Features
- **Shared Workspaces**: Collaborate with friends, family, or colleagues on tasks and projects.
- **Daily Agenda**: Keep track of your daily tasks and deadlines.
- **Project Management**: Organize tasks into projects and track progress.
- **Overdue Tasks**: Easily identify and prioritize tasks that are overdue.
- **Scheduled Tasks**: Schedule repeating tasks and events.
